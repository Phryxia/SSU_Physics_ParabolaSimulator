유체 속 포물선 운동 시뮬레이션
SSU 컴퓨터학부 나반 권세규(20142302)

1. 개요
이 프로그램은 유체 속에서의 포물선 운동을 탐구할 수 있는 시뮬레이터입니다
Processing 2.0(=JAVA)으로 제작되었기 때문에, 데스크톱 JAVA를 설치하셔야 합니다.

Java 8 : https://java.com/ko/download/

본 프로그램은 0.001초의 시간 해상도로 아래와 같은 시뮬레이션을 제공합니다.

- 진공 상태에서의 물체 운동 : 중력 가속도
- 공기가 있는 상태에서의 물체 운동 : 중력 가속도 + 항력
- 물 속에서의 물체 운동 : 중력 가속도 + 항력 + 부력

또한 발사 후 1분 동안의 물체 궤적을 표현해줍니다.



2. 사용 방법
32비트 윈도우에선 application.window32의 실행파일을,
64비트 윈도우에선 application.window64의 실행파일을 실행시켜야 합니다.

Drag the Ball! 문구가 떠 있을 때 물체를 누르면 붉은 화살표가 생깁니다.
이 화살표의 방향과 크기를 설정하여 마우스를 때면, 물체가 초기 속도를 가지고 발사됩니다.
(사실 이 화살표는 스프링과 동일합니다만, 본 프로그램에서 중요한 부분은 아닙니다)

m 키 : 모드를 바꿉니다. (진공 -> 공기 -> 물) 바뀐 모드는 화면 오른쪽 위에 표시됩니다.
        공기 모드나 물 속 모드일 경우 유체 흐름을 보여주는 화살표가 생깁니다.
SPACE 키 : 공을 제자리에 놓습니다.
좌우 화살표 키 : 유체 흐름의 방향을 조정합니다. 푸른 화살표의 방향이 바뀝니다.
상하 화살표 키 : 유체 흐름의 속력을 조정합니다. 푸른 화살표의 길이가 바뀝니다.

휠 : 물체의 질량을 바꿉니다. 부피는 자동으로 계산됩니다.
CTRL + 휠 : 물체의 밀도를 바꿉니다. 부피는 자동으로 계산됩니다.

*물체는 완전한 구형으로 가정하고 계산합니다. 시뮬레이션은 2D 상에서 이루어지지만,
 계산은 3차원 공간의 한 평면으로 가정하고 합니다.



3. 원리
시뮬레이션 상의 시간은 현실과 약간 다릅니다. 컴퓨터는 이산시간 공간이기 때문에 오차가 발생합니다.
시뮬레이션이 한 번 돌아가는 것을 1 틱이라고 합니다. 시뮬레이션은 한 틱 당 한 번의 위치 계산을 합니다.
한 틱과 다음 틱 사이의 시간 간격(dt)이 작으면 작을수록, 운동 오차가 줄어들게 됩니다.

틱과 틱 사이의 시간동안은 등가속도 운동을 한다고 가정하고 속도와 위치 벡터를 연산합니다.
매 틱마다 물체에 작용하는 합력을 계산한 뒤, 가속도 벡터로 변환시킵니다.

이 프로그램은 1초에 1000틱을 계산하며, 현실 시간과 1:1 비율로 시뮬레이션 됩니다.
(다만 컴퓨터 사양에 따라 다소 버벅거릴 수 있으며, 이로 인해 dt가 커져 오차가 발생할 수 있습니다.
